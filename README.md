# fizz-buzz

## How to test

* Go into example folder
* Hit "npm install"
* Hit "npm start" this will open browser at localhost:3000

## Running tests

* In root folder
* Hit "npm run test"
* You might want to run "npm run test -- --coverage" to see coverage stats
